# Knight's Tour Solver 
A web application for visualizing solutions to the knight's tour problem on nxn boards, using Warnsdorff's Rule to find solutions.

**Link:** https://knights-tour-solver.onrender.com/

# Built With
* React.js
* Vite
* Bootstrap.css
* Flask
